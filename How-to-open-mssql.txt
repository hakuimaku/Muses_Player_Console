How-to-open-mssql

Cách để mở mssql localhost server để có database cho app Muses_Player_Console

B1: Giải nén file mssql_shared_backup.tar.gz ở bất kỳ

B2: Mở terminal cd tới thư mục mssql_shared (chứa file docker-compose.yml) và gõ: docker compose up -d

B3:  Kiểm tra đã mở server chưa: docker ps

Có thể vào app và kiểm tra ngay đã đăng nhập được chưa
